ziggy-karma
====

a karma plugin for [ziggy](https://github.com/jarofghosts/ziggy)

## usage

#### !m, !merit, !thanks `<name>`

add a karma point to `<name>`

#### !dm, !demerit `<name>`

remove a karma point from `<name>`

#### !k, !karma `<name>`

check the karma points for `<name>`
